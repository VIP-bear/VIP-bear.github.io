KVM安装SSR及bbr

发表于 2018-03-10 | 阅读次数:

安装bbr

系统：centos-6-x86_64-bbr
进入服务器

安装wget:

yum -y install wget

然后执行以下代码：

wget --no-check-certificate https://github.com/teddysun/across/raw/master/bbr.sh

chmod +x bbr.sh

./bbr.sh

安装完成后可能会重启，需要重新登陆服务器。

安装SSR

wget -N --no-check-certificate   https://raw.githubusercontent.com/91yun/shadowsocks_install/master/shadowsocksR.sh && bash shadowsocksR.sh

安装过程中会要求你设置密码，然后选择一个远程端口，选好之后按下回车键，然后等待几分钟，
完成安装，最后会生成一张表格，记住红色部分。

下载客户端，就可以科学上网了。
